# Portfolio Starter for GitHub Pages

This is a simple static portfolio site you can publish at:

```text
https://SirTaxu.github.io
```

## Customize it

1. Replace `SirTaxu`, `ST`, and `you@example.com` in `index.html`.
2. Replace the project cards with your real projects.
3. Update the GitHub and LinkedIn links in the contact section.
4. Edit colors, spacing, and layout in `styles.css`.

## Publish on GitHub Pages

1. Create or open the GitHub repository named `SirTaxu.github.io`.
2. Push these files to that repository.
3. In GitHub, open the repository settings and go to **Pages**.
4. Set the source to deploy from a branch, usually `main`.
5. Visit `https://SirTaxu.github.io` after GitHub finishes publishing.

GitHub says Pages changes can take up to 10 minutes to publish.

Official guide: https://docs.github.com/en/pages/quickstart

## Optional command line setup

```powershell

git commit -m "Create portfolio starter"
git branch -M main
git remote add origin https://github.com/SirTaxu/SirTaxu.github.io.git
git push -u origin main
```
