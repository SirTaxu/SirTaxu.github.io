Qué garantiza la herramienta
Todos los archivos resultantes usan el mismo ancho y alto. Solo se copian píxeles; no se redimensiona ni interpola el asset. El modo recomendado conserva las coordenadas relativas que cada pose ya tenía dentro de su celda. El informe JSON registra cuadrícula, bounding boxes y posiciones de salida para poder auditar el proceso.
Procesamiento 100% local: ninguna imagen se envía fuera del navegador. Volver a SirTaxu.github.io
